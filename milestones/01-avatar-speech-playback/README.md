# Milestone 1: Avatar Speech Playback

## What it does
Avatar speaks with baseline lip sync linked to speech playback.

## Primary project areas
- apps/web/src/components/AvatarPlayer.tsx
- apps/web/src/components/Avatar3D.tsx
- apps/web/src/App.tsx
- apps/web/src/api/broadcastApi.ts

## Dependencies
- three
- @react-three/fiber
- @react-three/drei

## Expected output
- 3D avatar renders
- audio playback works
- basic mouth movement during speech
